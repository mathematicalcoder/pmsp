Pisay Mathers Summer Program

The Pisay Mathers Summer Program aims to help aspiring math olympians in PSHS prepare for the Philippine Mathematical Olympiad before the school year starts. The program will run from 17 June 2024 to 16 August 2024.

Note that this website is still under construction.
